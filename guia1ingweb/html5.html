HTML5 y la estructura semántica

HTML5 introdujo etiquetas semánticas como <header>, <nav>, <main>, <section>, <article> y <footer> que permiten organizar el contenido de manera lógica y comprensible tanto para los desarrolladores como para los motores de búsqueda y lectores de pantalla.

Estas etiquetas mejoran la accesibilidad, el posicionamiento SEO y el mantenimiento del código. Usarlas correctamente es una buena práctica en cualquier proyecto web moderno.
