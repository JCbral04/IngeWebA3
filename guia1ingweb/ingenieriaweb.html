¿Qué es la Ingeniería Web?

La ingeniería web es una rama de la ingeniería de software que se enfoca en el desarrollo disciplinado de aplicaciones web, utilizando metodologías y herramientas que garanticen calidad, escalabilidad, seguridad y buena experiencia de usuario.

Esta disciplina combina conocimientos de desarrollo frontend, backend, arquitectura web, usabilidad, control de versiones, accesibilidad, y pruebas.

Recursos recomendados:

  • Guía de desarrollo web en MDN Web Docs
  • FreeCodeCamp (cursos gratuitos)
  • Google Web.dev: Aprende desarrollo web
